# Цикли 

При написанні скриптів часто постає завдання зробити однотипні дії багато разів.

Наприклад, вивести товари зі списку один за одним чи просто запустити один і той же код для кожного числа від 1 до 10.

Цикл – це спосіб повторити один і той же код кілька разів.

## Цикл for
Цикл ```for``` повторює дії, доки не відбудеться будь-яка спеціальна подія завершення циклу. Оператор ```for``` JavaScript аналогічний оператору ```for``` Java і C. Оголошення оператора ```for``` виглядає наступним чином:

<CodeEdit text={`
for ([початок]; [умова]; [крок]) { вираз }
`}/>

При його виконанні відбувається таке:

1. Виконується ```вираз``` початок, якщо вона вказана. Цей вираз зазвичай ініціалізує один або кілька лічильників, але синтаксис дозволяє ```виразу``` будь-якої складності. Також використовується для оголошення змінних.
2. Виконується ```умова```. Якщо ```умова``` істинна (true), то виконуються ```вирази```. Якщо воно хибне (false), цикл ```for``` переривається. Якщо ж умова повністю пропущено, воно вважається істинним.
3. Виконуються ```вирази```. Щоб виконати кілька ```виразів```, використовується блок-вираз ```{ ... }``` для угруповання ```виразів```.
4. Оновлюється ```крок```, якщо він є, а потім керування повертається до кроку ```2```.

#### Приклад:
<CodeEdit text={`
let i = 0;
for (let i = 0; i < 3; i++) { // Виводить 0, далі 1, потім 2
  console.log(i);
}
`}/>

## Цикл while
Цикл ```while``` виконує вирази поки умова істинна. Виглядає він так:

<CodeEdit text={`
while (умова) { вирази }
`}/>

Якщо умова стає хибною, ```вирази``` в циклі перестають виконуватися і управління переходить до ```виразу``` після циклу.

Умова перевіряється на істинність доти, як виконуються висловлювання у циклі. Якщо ```умова``` є істинною, виконуються ```вирази```, а потім ```умова``` перевіряється знову. Якщо ```умова``` хибна, виконання зупиняється і керування переходить до ```виразу``` після ```while```.

#### Приклад: 

<CodeEdit text={`
let i = 0;
while (i < 3) { // показується 0, далі 1, потім 2
  alert( i );
  i++;
}
`}/>

## Цикл do...while
Цикл ```do...while``` повторюється поки що задана умова істинна. Оператор ```do...while``` має вигляд:

<CodeEdit text={`
do{
  вирази
}
while (умова);
`}/>

```вирази``` виконуються поки ```умова``` істинна. Щоб використати кілька ```виразів```, використовуйте блок-вираз ```{ ... }```, щоб згрупувати їх. Якщо умова істинна, ```вирази``` виконаються знову. Наприкінці кожного проходу ```умова``` перевіряється. Якщо ```умова``` хибна, виконання зупиняється і управління передається ```виразу``` після ```do...while```.

#### Приклад:

<CodeEdit text={`
let i = 0;
do {
  alert( i );
  i++;
} while (i < 3);
`}/>

## Переривання циклу: “break”
Зазвичай, цикл завершується, коли умова стає ```false```.

Але ми можемо в будь-який момент вийти з циклу, використавши спеціальну директиву ```break```.

Наприклад, наступний код запитує в користувача число до тих пір, поки користувач їх вводить. Після того, як користувач не ввів число — цикл завершується (директивою “break”) і виводить суму чисел:

<CodeEdit text={`
let sum = 0;

while (true) {

  let value = +prompt("Введіть число", '');

  if (!value) break; // (*) Вихід з циклу

  sum += value;

}
alert( 'Сума: ' + sum );
`}/>

## Продовження з наступної ітерації (continue)
Директива ```continue``` — це “полегшена версія” ```break```. Вона не зупиняє весь цикл. Натомість, вона зупиняє поточну ітерацію і починає виконання циклу спочатку з наступної ітерації (якщо умова циклу досі вірна).

Її зручно використовувати коли закінчили з поточною ітерацією і хочемо продовжити з наступної.

Цикл нижче використовує continue щоб вивести лише непарні значення:

<CodeEdit text={`
for (let i = 0; i < 10; i++) {

  // якщо умова справджується, тоді пропускаємо решту тіла циклу і починаємо з наступної ітерації
  if (i % 2 == 0) continue;

  alert(i); // 1, потім 3, 5, 7, 9
}
`}/>
